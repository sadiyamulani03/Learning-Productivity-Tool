# 📊 Learning Productivity Intelligence Tool

A standalone desktop application built using Python and MySQL to analyze and improve student learning productivity.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register with bcrypt)
- 📚 Subject Management
- ⏱ Study Session Tracking (focus, duration, difficulty)
- 📈 Productivity Analytics (visual insights using Matplotlib)
- 🧠 Fatigue Detection based on learning patterns
- 🎯 Goal Management with completion tracking
- 🔔 Notification System (alerts & reminders)
- 📄 Report Generation (CSV export)
- ⚙️ Settings Module (theme, reminders, preferences)

---

## 🛠 Tech Stack

- Python (Tkinter for GUI)
- MySQL (Relational Database)
- Matplotlib (Data Visualization)

---

## 🧩 System Architecture

User → UI (Tkinter) → Python Logic → MySQL Database  
→ Data Processing → Analytics → Insights & Reports
